# Проект: [Предсказание стоимости жилья в Калифорнии с использованием Apache Spark]()
## Описание проекта
Проект посвящён задаче предсказания медианной стоимости жилья в Калифорнии. Для обработки данных и обучения моделей используется фреймворк Apache Spark и библиотека MLlib.

## Цель проекта
Построить и сравнить модели линейной регрессии для предсказания медианной стоимости жилья в Калифорнии с использованием Apache Spark на основе полного набора признаков и только числовых признаков.

## Результаты
* Создан единый пайплайн предобработки данных в Apache Spark MLlib с заполнением пропусков, кодированием категориальных признаков и масштабированием числовых данных.
* Обучены две модели линейной регрессии: на полном наборе признаков и только на числовых.
* Модель с полным набором признаков показала лучшие метрики (RMSE, MAE, R²) по сравнению с моделью на числовых признаках.
* Линейная регрессия даёт приемлемое качество прогнозирования, но не полностью учитывает сложные зависимости в данных.

## Рекомендации
* Для улучшения качества прогнозирования попробовать использование нелинейных моделей, способных учитывать сложные зависимости между признаками.

## Приобретенные навыки
* Работа с Apache Spark и MLlib.
* Создание пайплайнов обработки данных в MLlib.
* Обучение и подбор гиперпараметров моделей линейной регрессии в MLlib.
* Интерпретация метрик регрессионных моделей.
