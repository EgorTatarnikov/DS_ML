## Описание проекта
Проект посвящён анализу данных сервиса аренды самокатов GoFast. Исследуется поведение пользователей, их поездки и условия подписки, чтобы выявить закономерности и определить факторы, влияющие на доходность сервиса.

## Цель проекта
Проверить гипотезы о поведении пользователей с подпиской и без неё, оценить их вклад в выручку и выработать рекомендации для увеличения числа подписчиков.

## Результаты
* Пользователи с подпиской проводят больше времени на поездках.
* Среднее расстояние поездок подписчиков не превышает оптимального значения для износа самокатов.
* Помесячная выручка от подписчиков выше, чем от пользователей без подписки.

## Рекомендации
* Продвигать подписку среди пользователей без подписки, чтобы увеличить доход и вовлечённость.
* Проверить корректность фиксации времени поездок.
* Исключить пользователей младше 18 лет для соблюдения законодательства и норм проката.
