# Проект: [Прогнозирование удовлетворённости и увольнения сотрудников]()
## Описание проекта
Проект посвящён анализу HR-данных. Разрабатывались модели машинного обучения для прогнозирования уровня удовлетворённости сотрудников и вероятности их увольнения.

## Цели проекта
Разработать модели для:
* Прогнозирования удовлетворённости сотрудников работой.
* Определения вероятности увольнения сотрудников.

## Результаты
1. Прогноз удовлетворённости:
* Модель: RandomForestRegressor (SMAPE = 14,07%)
* Важнейшие факторы: оценка руководителя, стаж, должность, зарплата, загруженность
2. Прогноз увольнения:
* Модель: RandomForestClassifier (ROC-AUC = 0,92)
* Важнейшие факторы: должность, стаж, удовлетворённость, зарплата, оценка руководителя, загруженность

## Рекомендации
* Регулярная оценка сотрудников и корректировка зарплат/бонусов
* Оптимизация рабочих нагрузок (средняя нагрузка снижает текучесть)
* Программы карьерного роста, обучения и наставничества
